Northwind Database Schema

Full Database

Complete overview of the Northwind database structure with all tables and their key columns.

📦 Categories

  • CategoryID - Primary Key
  • CategoryName - Category name
  • Description - Category description
  • Picture - Category image

👥 Customers

  • CustomerID - Primary Key (5 chars)
  • CompanyName - Company name
  • ContactName - Contact person
  • ContactTitle - Contact title
  • Address, City, Region, PostalCode
  • Country - Customer location
  • Phone, Fax - Contact info

👔 Employees

  • EmployeeID - Primary Key
  • LastName, FirstName - Employee name
  • Title - Job title
  • TitleOfCourtesy - Mr./Ms./Dr.
  • BirthDate, HireDate - Dates
  • Address, City, Region, PostalCode, Country
  • HomePhone, Extension
  • Photo, Notes - Additional info
  • ReportsTo - Manager ID
  • PhotoPath - Photo location

📋 Orders

  • OrderID - Primary Key
  • CustomerID - FK to Customers
  • EmployeeID - FK to Employees
  • OrderDate - When ordered
  • RequiredDate - When needed
  • ShippedDate - When shipped
  • ShipVia - Shipper ID
  • Freight - Shipping cost
  • ShipName, ShipAddress, ShipCity
  • ShipRegion, ShipPostalCode, ShipCountry

📝 Order Details

  • OrderID - FK to Orders (PK)
  • ProductID - FK to Products (PK)
  • UnitPrice - Price per unit
  • Quantity - Number ordered
  • Discount - Discount percentage

🏭 Products

  • ProductID - Primary Key
  • ProductName - Product name
  • SupplierID - FK to Suppliers
  • CategoryID - FK to Categories
  • QuantityPerUnit - Package info
  • UnitPrice - Current price
  • UnitsInStock - Current inventory
  • UnitsOnOrder - Pending orders
  • ReorderLevel - Reorder threshold
  • Discontinued - Active status

🚚 Shippers

  • ShipperID - Primary Key
  • CompanyName - Shipper name
  • Phone - Contact number

🏢 Suppliers

  • SupplierID - Primary Key
  • CompanyName - Supplier name
  • ContactName - Contact person
  • ContactTitle - Contact title
  • Address, City, Region, PostalCode
  • Country - Supplier location
  • Phone, Fax - Contact info
  • HomePage - Website URL

🌍 Region

  • RegionID - Primary Key
  • RegionDescription - Region name

📍 Territories

  • TerritoryID - Primary Key
  • TerritoryDescription - Territory name
  • RegionID - FK to Region

🗺️ EmployeeTerritories

  • EmployeeID - FK to Employees (PK)
  • TerritoryID - FK to Territories (PK)

📊 CustomerDemographics

  • CustomerTypeID - Primary Key
  • CustomerDesc - Description

🔗 CustomerCustomerDemo

  • CustomerID - FK to Customers (PK)
  • CustomerTypeID - FK to CustomerDemographics (PK)
